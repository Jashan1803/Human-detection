# Human Detection

A machine learning-based system for detecting humans in images and video streams using YOLO v8.

## 📊 Features

- Real-time human detection in images
- Batch processing of multiple images
- Web interface for easy usage
- Command-line interface for developers
- High accuracy with YOLO v8 model
- Supports multiple image formats (JPG, PNG)
